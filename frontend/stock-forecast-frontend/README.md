# Stock Forecast Frontend

## Prerequisites
- Node.js (v18+)
- npm

## Installation
1. Install dependencies:
```bash
npm install
```

## Development
Run the development server:
```bash
npm run dev
```

## Build
Create a production build:
```bash
npm run build
```

## Technologies
- Next.js
- React
- Tailwind CSS
- Chart.js
- Axios
